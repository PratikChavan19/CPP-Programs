# C++ Programs 🚀

A collection of C++ programs developed to understand and practice the fundamental concepts of **C++ programming**, **Object-Oriented Programming**, **Standard Template Library (STL)**, and **problem-solving techniques**.

This repository serves as a personal learning resource and reference guide, containing implementations that demonstrate various C++ concepts ranging from basic programming constructs to advanced language features.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The objective of this repository is to:

* Understand Object-Oriented Programming concepts
* Explore the Standard Template Library (STL)

## ▶️ Getting Started :

### Clone the Repository

```bash
git clone https://github.com/PratikChavan19/CPP-Programs.git
cd CPP-Programs
```

### Compile a Program

Using **G++**:

```bash
g++ ProgramName.cpp -o ProgramName
```

### Run the Program

Linux/macOS:

```bash
./ProgramName
```

Windows:

```bash
ProgramName.exe
```
